# The Bad Parts

* Global Variables
* Semicolon insertion
* \+ adds and concatenates
* `typeof`
* `with` and `eval`
* phony arrays
* `false`, `null`, `undefined`, `NaN`
* `==` and `!=`

---

# The Good Parts

* Loose Typing
* Dynamic Objects
* Object Literals
* Functions (lambda)
